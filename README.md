Problem #31

Given any infinite set $A\subset \mathbb{N}$ there is a set $B$ of density $0$ such that $A+B$ contains all except finitely many integers.

Conjectured by Erd\H{o}s and Straus. Proved by Lorentz \cite{Lo54}.
